# movinghorse.github.io

这是 `movinghorse` 的 GitHub Pages 用户主页仓库，对应的访问地址为：

- `https://movinghorse.github.io`

## 仓库结构

- `index.html`：主页 HTML 文件（关于我页面，可以按需修改文案）
- `assets/css/style.css`：站点样式文件
- `_config.yml`：GitHub Pages / Jekyll 的基础配置（站点标题、描述等）

## 使用方式

1. 确保 GitHub 仓库名为 `movinghorse.github.io`，分支为 `main`（或在 Pages 设置中选择对应分支和路径为 `/（root）`）。
2. 将本地代码推送到 GitHub：

   ```bash
   git add .
   git commit -m "Initialize GitHub Pages site"
   git push origin main
   ```

3. 在 GitHub 仓库的 **Settings → Pages** 中确认：
   - **Source** 选择 `Deploy from a branch`
   - **Branch** 选择 `main` 和 `/ (root)`

几分钟后即可通过 `https://movinghorse.github.io` 访问你的个人主页。
